# 📈 PaperTrader

Ever wanted to try investing but didn't want to risk real money? PaperTrader lets you buy and sell real stocks with $100,000 of fake cash using live market data — no risk, all the fun.

🚀 **[Try it live here](https://papertrader.streamlit.app)**

---

## What it does

- 🔍 **Search any stock** by name or ticker and get live prices instantly
- 📊 **Interactive price charts** with period selectors (1D, 1W, 1M, 6M, 1Y, 5Y)
- 💸 **Buy and sell stocks** and track your portfolio in real time
- 🏠 **Dashboard** showing your total value, cash balance, holdings breakdown, and a pie chart of your positions
- 📋 **Transaction history** with ticker filtering
- 📉 Tracks your **P&L, ROI, cost basis, and unrealized gains** per stock

## Run it locally

1. Clone the repo
```bash
git clone https://github.com/Iamaradia/Paper-Trader.git
cd Paper-Trader
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the app
```bash
streamlit run GUI.py
```

## How the trading engine works

The trading engine is built from scratch — no libraries, just Python and CSV files. It tracks:
- **Cost basis** — what you paid on average per share
- **Unrealized P&L** — profit/loss on shares you still own
- **Realized P&L** — profit/loss on shares you've sold
- **ROI** — your return on investment as a percentage
